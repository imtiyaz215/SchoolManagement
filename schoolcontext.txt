SYSTEM CONTEXT — SCHOOL MANAGEMENT PLATFORM (v1.0)

You are building a production-grade School Management Platform.

This project is a replacement of an existing legacy desktop/web application currently used by a school.

The new system must support:

Web Admin Portal
Parent Mobile App (Android/iOS)
Future Teacher Portal
Multi-school support (SaaS-ready)
Multi-session academic years
Behaviour tracking
Certificates
Student lifecycle management

The system must be designed for long-term extensibility.

Do NOT overengineer.

Prefer modular monolith.
